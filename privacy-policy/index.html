隐私政策

数据收集和使用

本扩展收集以下数据：

  • 搜索引擎偏好设置
  • 背景图片设置
  • 时钟显示格式设置
  • 天气显示设置
  • 位置信息（仅用于天气功能，详见下文）

所有数据仅存储在用户本地，不会上传至任何服务器。

位置信息处理

当您启用天气功能时，扩展需要访问您的位置信息以提供准确的天气预报：

  • 位置数据仅用于通过高德地图API获取天气信息
  • 我们不会记录或存储您的精确位置历史
  • 位置信息仅在必要时发送给高德地图API，且通过HTTPS加密传输
  • 您可以随时通过关闭天气功能来停止位置信息的使用

第三方服务

本扩展使用以下第三方服务：

  • 默认搜索引擎：使用Chrome Search API
  • Google搜索
  • Bing搜索
  • 百度搜索
  • DuckDuckGo搜索
  • Yahoo搜索
  • Yandex搜索
  • 小小API（用于获取背景图片）
  • Unsplash API（用于获取背景图片）
  • Pixabay API（用于获取背景图片）
  • Pexels API（用于获取背景图片）
  • 高德地图API（用于获取天气信息）

数据存储细节

关于您的数据如何存储：

  • 所有设置使用Chrome提供的chrome.storage.localAPI存储在本地
  • 天气数据会临时缓存（最长24小时）以减少API调用
  • 扩展不会设置任何cookie
  • 您的数据会一直保存到您卸载扩展或手动清除数据

数据安全

  • 所有网络请求均使用HTTPS加密传输
  • API密钥经过加密存储
  • 不收集用户的个人识别信息

数据权限

用户可以随时：

  • 清除所有存储的设置
  • 关闭天气功能
  • 更改或删除API密钥
  • 通过浏览器扩展管理页面撤销位置权限

未成年人保护

本扩展适合所有年龄段用户使用，不会收集未成年人的个人信息。我们鼓励家长和监护人参与未成年人的在线活动，确保他们的隐私得到保护。

政策变更

我们可能会不时更新本隐私政策。当有重大变更时，我们会在扩展更新时通知您。继续使用本扩展即表示您接受修订后的隐私政策。
